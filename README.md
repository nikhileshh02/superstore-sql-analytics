# Super Store Retail Analytics — Advanced SQL
## MySQL | 23 Advanced Queries | Window Functions

## Project Overview
Comprehensive SQL analytics on a US retail dataset covering sales, profit,
customers, and shipping across all product categories and regions.

## Queries Included
**Basic Analytics:**
- Total Sales by Region | Profit Margin by Category
- Monthly Sales Trends | Discount Distribution
- Sales by Subcategory | Quantity by State

**Advanced Analytics:**
- Sales Growth Rate using LAG() window function (month-over-month %)
- Customer Lifetime Value (CLV)
- Average Order Value (AOV)
- Seasonal Sales Patterns (Spring/Summer/Autumn/Winter using CASE WHEN)
- High-Revenue Low-Margin Products (HAVING with subquery)
- Average Shipping Time using DATEDIFF()
- Repeat Purchase Rate

## Key SQL Concepts Used
GROUP BY + HAVING | ORDER BY + LIMIT | CASE WHEN | LAG() Window Function
DATEDIFF() | MONTHNAME() | ROUND() | Subqueries | CTEs
